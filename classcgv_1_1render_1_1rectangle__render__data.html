Detailed Description

template<typename ColorType = rgb>
class cgv::render::rectangle_render_data< ColorType >

Render data for rectangle geometry with support for the rectangle_renderer.

See render_data_base.

Template Parameters
ColorTypeThe type used to represent colors. Must be cgv::render::rgb or cgv::render::rgba.

Draw the stored geometry using the given renderer.

This method must only be called after a successful call to enable. Afterwards, disable must be called. If offset and count are not specified the full data range is rendered. Validates whether offset and count produce a valid range according to the value returned by render_data_base::render_count and clamps the count values if necessary.

◆ set_out_of_date()

void cgv::render::render_data_base< rgb >::set_out_of_date ( )
inlineinherited

Notify the render data about state changes.

Calling this method will result in the state_out_of_date flag being set to true and a call to transfer upon the next call to render or enable. This method shall never be called implicitly as a side effect of another member method, except clear, and must always be called explicitly after changing the stored data.
