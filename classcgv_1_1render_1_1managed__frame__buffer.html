bool cgv::render::managed_frame_buffer::set_size ( const ivec2 &  size)

Sets the size of the framebuffer renderbuffers.

Set the x and/or y-component to <= 0 to use the width/height from the context. Returns false if the size is larger than the maximum allowed render buffer size, as given by the OpenGL API.
